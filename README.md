# Spendy — трекер расходов

Личный трекер расходов на чистом JavaScript (ES6+) с хранением данных в `localStorage`.
Тёмно-синяя тема, градиенты, стекло (glassmorphism), анимированные графики.

## Возможности

- Регистрация и вход (аккаунты хранятся локально, расходы привязаны к пользователю)
- Приветствие по имени на главной
- Добавление, редактирование и удаление расходов (с подтверждением)
- Отметка «оплачено / запланировано» (чекбокс)
- Фиксированные категории + создание собственных
- Переключение валюты: ₸ / $ / ₽
- Поиск по названию и фильтры по категории и статусу
- Статистика с анимированными графиками за день / неделю / месяц / 6 / 12 месяцев
- Адаптивная вёрстка (от 320px)

## Структура

```
index.html        — главная (вход/регистрация + список расходов)
form.html         — добавление / редактирование расхода
about.html        — о проекте и авторе
stats.html        — статистика и графики
css/style.css     — стили
js/storage.js     — работа с localStorage
js/utils.js       — валюты, категории, форматирование
js/auth.js        — авторизация, шапка, навигация
js/main.js        — логика главной
js/form.js        — логика формы
js/stats.js       — логика статистики
images/           — иконки
```

## Технологии

HTML5 · CSS3 (Flexbox, Grid) · JavaScript (ES6+) · localStorage

## Запуск

Откройте `index.html` в браузере или опубликуйте через GitHub Pages.
